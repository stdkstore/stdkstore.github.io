Our custom OS: II

II (pronounced 'I I' or 'Two') is a simple, cli based operating system built from the ground up using Assembly and C. It's engineered for maximum performance, reliability, and security.

This is a custom kernel, not based on Linux, Windows, or any existing distribution. We eliminated all bloatware and unnecessary complexity, providing a clean, efficient environment suitable for both technical professionals and beginners who value pure computing.

Current Focus (ii25 release):

  • Built from scratch (custom kernel)
  • Zero bloatware, minimal resource usage
  • Currently Command-Line Interface (CLI) only
  • Simple design, safe and unbreakable
