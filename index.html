What II25 is

II25 is a brand‑new, command‑line operating system currently in active development. It’s built entirely from scratch using Assembly and C, with a custom kernel, no bloatware, and an obsessive focus on simplicity, safety, and performance.

It’s currently CLI‑based and intentionally small — made for people who want to see the entire stack, audit what’s running, and keep their systems understandable.

Current focus (II25 release)

  • Bootable ISO image (VirtualBox / VMware / QEMU ready)
  • Custom kernel, built from first principles
  • Zero bloatware, minimal memory + disk footprint
  • Command-Line Interface only
  • Simple design, professional foundations

System requirements

II25 is intentionally lightweight — it runs comfortably in tiny virtual machines and is even runnable directly in your browser via copy.sh.

Platform Tested on Linux VMs (VirtualBox / QEMU)
RAM 64 MB minimum • 128 MB recommended
Disk ~10 MB
CPU 1 core (multi‑core support planned)
Browser Runnable via copy.sh
