II25 by IIntegre

II25 is an experimental, brand-new command-line operating system currently in active development. Built entirely from scratch using Assembly and C, it prioritizes safety, simplicity, and performance.

Custom kernel, zero bloatware. Currently CLI-based and designed for people who actually care about what's running on their machine.

Current Focus (II25 release):

  • Available as an ISO image (works in VirtualBox/VMware)
  • Built from scratch (custom kernel)
  • Zero bloatware, minimal resource usage
  • Currently Command-Line Interface (CLI) only
  • Simple design, professional results

System Requirements:

  • Tested on Linux (VirtualBox / QEMU)
  • RAM: Minimum 64MB (works best with 128MB)
  • Disk size: ~10MB
  • CPU: 1 core (multi-core support coming later)
